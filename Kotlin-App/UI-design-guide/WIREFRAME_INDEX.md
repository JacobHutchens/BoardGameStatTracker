## Wireframe Description Format

Each wireframe description includes:

1. **Screen Layout**: ASCII diagram showing component placement
2. **Component Specifications**: Detailed specs for each component
3. **Spacing & Layout**: Exact spacing and layout rules
4. **States & Interactions**: All possible states and interactions
5. **Accessibility**: Screen reader support, touch targets, contrast
6. **Navigation**: All navigation paths from this screen
